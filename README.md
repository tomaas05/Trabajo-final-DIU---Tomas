---

## Descripción

Propuesta de rediseño web para el **EcoMercado UGR**, el mercado agroecológico 
universitario del Campus Fuentenueva de la Universidad de Granada. Este repositorio 
contiene el prototipo de alta fidelidad (mockup) desarrollado como parte del caso 
de estudio de la asignatura DIU 2025/26.

El diseño responde a los *insights* extraídos del análisis de usabilidad, 
accesibilidad y experiencia de usuario de la web de referencia 
[Mercados de Campesinos](https://xarxaconsum.org/es/mercados-de-campesinos/), 
aplicando principios de Diseño Centrado en el Usuario (DCU), evaluación 
heurística (Nielsen), WCAG 2.2 y metodología Atomic Design.

## Decisiones de diseño destacadas

| Principio | Aplicación en el mockup |
|---|---|
| *Mobile-first* | Layout en columnas flexibles desde 360 px; touch targets ≥ 44×44 px |
| Accesibilidad WCAG 2.2 AA | Contraste ≥ 4.5:1, semántica HTML5, atributos ARIA, skip-link |
| Atomic Design | Tarjeta de mercado como componente reutilizable (organismo) |
| UX Writing | CTAs con verbos de acción: *Ver en el mapa*, *Hacer mi pedido* |
| Tipografía dual | Playfair Display (display) + Inter (cuerpo), cuerpo mínimo 16 px |
| Paleta UGR | Verde institucional `#00703C` como color base (regla 60-30-10) |
